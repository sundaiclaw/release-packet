# Proposal: Release Packet Launch

## Why
Shipping is messy. Teams have PR notes, issue links, screenshots, metrics, and half-written status updates scattered everywhere. Release Packet turns that rough input into a clean, decision-ready launch packet.

## What Changes
- Build a single-page app where users paste rough release notes, links, and evidence.
- Add AI generation for a release summary, changelog, QA checklist, and launch copy.
- Add a risk audit section that highlights missing proof and likely launch gaps.

## Capabilities
- New capability: `release-drafting`
- New capability: `risk-audit`
- New capability: `launch-assets`
